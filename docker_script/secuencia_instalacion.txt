# TIPS

Las imagenes ampliadas tienen las bibliotecas necesarias para el correcto funcionamiento del codigo fuente (ver imagenes.txt)
Se procura que todo el codigo fuente tenga un respaldo siempre actualizado en GITHUB y BitBucket
Se procura que la informacion del modelo de base de datos, procedimientos, funciones, tiggers y datos tengan respaldo en GITHUB y BitBucket
Lo anterior hace que los contenedores sean "deshechables", ya que se pueden regenerar en cualquier momento
